Data Types

C supports the following basic data types:

int : integer, a whole number

float: a floating point, a number with fractional part

double: double-precision floating point value,

char: single character.


Variables

A variable is the name for an area in the memory.

The name of a variable ( also called the identifier) must begin with either a letter or an underscore and can be composed of letters, digits and underscore character.


Example: the following statement declares an integer variable var and then assigns it the value 20.

int var = 20;  


Constants

A constant stores a value that can not be changed from its initial assignment.

By using the constant with meaningful names, code is easier to read and understand.

One way to declare a constant is by using the const keyword in the variable declaration:


#include <stdio.h>

int main() {
   const double PI = 3.14;
   printf ( "%f", PI ) ; 
   return 0;
}

The value of PI cannot be changed during the program execution.


Another way to define  a constant is with #define pre-processor directive.

#include <stdio.h>
#define PI 3.14
int main() {
   
   printf ( "%f", PI ) ; 
   return 0;
}
