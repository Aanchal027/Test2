Formatting Input

C supports a number of ways for taking user input.

The scanf() function is used to assign input to variables. A all to this function scans input according to format specifier that convert input as necessary.

If input can't be converted, then the assignment isn't made.The scanf() statement waits for input and then makes assignment:

int x;
float num;
char text[20];
scanf ("%d %f  %s", &x, &num, text );

Typing  a numeric value like 20, floating point value like 2.5 and a string such as "abcd" will be assigned to  x , num and text. 

Note that & must be used to access the variable addresses. The & isn't needed for a string because a string name acts as a pointer. 

Format Specifier begin with a percent sign % and are used to assign values to corresponding arguments after control string, Blanks, tabs and newlines are ignored. Following are used with specifier for input:

  • d decimal
  • c character
  • s string
  • f float
  • x hexadecimal
