Class is a collection of data members and member functions under one name.

or

A class can be defined as a template/blueprint that describes the behaviour/state that the object of its type support.

Example:

public class Dog {
   String breed;
   int age;
   String color;

   void barking() {
   }

   void hungry() {
   }

   void sleeping() {
   }
}

Object: an object of a class is an instance of that class exhibiting the properties of the respective class

or

Objects have states and behaviours. Example: A dog has states - color, name, breed as well as behaviours – wagging the tail, barking, eating.

Example:

public class Puppy {
   public Puppy(String name) {
      // This constructor has one parameter, name.
      System.out.println("Passed Name is :" + name );
   }

   public static void main(String []args) {
      // Following statement would create an object myPuppy
      Puppy myPuppy = new Puppy( "tommy" );
   }
}
