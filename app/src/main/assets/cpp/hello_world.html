Hello World Using C++

A C++ program is collection of commands or statements. 

Below is a simple code that has "Hello World!" as its output.


#include <iostream>
using namespace std;

// main() is where program execution begins.
int main() {
   // prints Hello World
   cout << "Hello World"; 
   return 0;
}

Lets Break The Program

C++ offers various headers, each of which contains information needed for programs to work properly. This particular programs call for the header <iostream>. The number sign (#) at the beginning of a line targets the compiler's pre-processor. In this case #include tells the pre-processor to include the <iostream> header.

Main

Program execution begins at int main(). Curly brackets{} indicate the beginning and end of a function which can also be called functions body, The information inside the bracket indicates what the functions does when executed
