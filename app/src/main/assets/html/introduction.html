What is HTML?

HTML stands for Hyper Text Markup Language.

Unlike a scripting or programming language that uses scripts to perform functions, a markup language uses tags to identify content.

Here is an  example of an html tag:

<p> I'm a paragraph </p>

The Web Structure

The ability to code using HTML is essential for any web professional. Acquiring this skill should be starting point for anyone who is learning how to create content for web. 

Modern Web Design

  • HTML : Structure
  • CSS : Presentation
  • JavaScript : Behavior
  • PHP or similar : Backend
  • CMS : Content Management
