The HTML File

Html files are text files, so you can use any text editor to create you first web page. There are some very nice HTML editors available; you can choose the one that works for your.

Add the basic HTML structure to the text editor with " This is a line of text" in the body section 

<html>
     <head></head>
     <body>
         This is a line of Text.
     </body>
</html>
